기능
- 로그인 화면:
	DB(동적 적용) - DB Table(최대 3개)
	-> 1. DB : power shell 확인
	-> 2. ping : Query(select 1) 확인

- 메인 화면:
	1. Text Log : 현재 wpf의 모든 로그 확인
	2. DB(tab) : DB Table 상태 확인(Color)
	3. 테스트 기능 : 
		-> 1. Auto Mode: Query(select 1) 2초간격 지속 확인
			- 주의) query 문이 잘못 된건지 db 접속이 끊긴 건지 확인 해야함
				- green: DB 주소, Query 둘다 양호
				- orange: DB 주소: 불량, Query: 양호
				- red: DB 주소: 불량, Query: 불량, DB Table 불량

		-> 2. Func Button: 수동 DB 상태 확인 (PS, Ping)
